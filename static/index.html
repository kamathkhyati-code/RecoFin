RecoFin

An agentic reconciliation system - it matches a company's books against the bank and flags what doesn't line up.

Pipeline

  1. Ingest - CSV upload or sample data
  2. Validate - catch bad data
  3. Normalize - one currency, canonical names
  4. Match - real matching agent: exact -> tolerance -> fuzzy
